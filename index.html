What time is it

A single webpage written entirely in javascript (no database needed).

Installation is easy as download package & open index.html in your favorite browser

Install

          
          
            > Download file
            > Unzip to local file system

            Open browser to page to: 
            file:///Users/kbusi/Devel/what-time-is-it/index.html

Usage

There are two clocks. The clock on left hand side shows your current location's time. The clock on right hand side shows target city's time. The default is Tampa, FL. Enter city in the input bar and time is updated. Simple!

For best results use the cities suggested by autocomplete
