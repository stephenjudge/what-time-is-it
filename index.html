What time is it in

Single web page digital clock

Download this project as a .zip file Download this project as a tar.gz file

Shows time in your city and another city of your choosing

written entirely in javascript (no database needed).

demo hosted on github

Install

  • Download file
  • Unzip to local file system
  • Open browser to time.html in the unzipped directory

Usage

There are two clocks. Clock on left and clock on right

Clock on left is your current location.

Clock on right is city where you want to know the time

Enter city in text box (use autocomplete) to select city you want to see the time in and hit enter.

Clock on right side should get updated automatically
